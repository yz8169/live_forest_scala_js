简介

欢迎使用 LiveForest， 本软件用于对 肝纤维化 和 肝硬化 的风险评估。

本软件适用的分析样本为18岁及以上的成年人样本，不包含孕妇样本。

我们构建了3个 随机森林模型(Random Forest，RF) 旨在: 1）诊断慢性肝病；2）区分肝纤维化和肝硬化；3）区分早期和晚期肝纤维化。

- 模型-1 用于诊断 慢性肝病 ；

- 模型-2 用于区分 肝纤维化 和 肝硬化 ；

- 模型-3 用于区分 早期肝纤维化 和 晚期肝纤维化 。

操作说明

预测：

  1. 点击左侧导航栏的“预测”分页；
  2. 在页面输入各项指标的值；
  3. 点击“运行”按钮；
  4. 页面下方会显示相应结果；
  5. 单击“导出报告”按钮下载pdf格式的预测报告。

批量预测：

  1. 点击左侧导航栏的“批量预测”分页；
  2. 在页面上传符合格式要求的.xlsx文件；
  3. 点击“运行”按钮；
  4. 点击"下载"按钮下载结果。
